🔒 Loopholes Arrested — Every Degree of Freedom Locked
WHERE FRAUD HAPPENS NOW → HOW DS BLOCKS IT
Broker alters RTC owner name
→ QR code on RTC verified against Bhoomi portal directly. Mismatch = FORGERY alert. Cannot alter a QR without the government's private key.
EC shows clean but hidden bank mortgage exists
→ CERSAI (RBI registry) checked simultaneously. EC + CERSAI contradiction = automatic DO NOT BUY. No single document trusted alone.
Broker takes cash advance and disappears
→ Advance paid through DS escrow. Held until verification clears. Auto-refunded if seller defaults. No cash changes hands outside platform.
Fake e-Stamp (reused or printed stamp paper)
→ SHCIL UIN verified. Duplicate UIN = FRAUD alert. Used stamps flagged in system. Impossible to reuse.
Court cases hidden by seller
→ eCourts searched by owner name (not just survey number). Finds cases even if not tied to the property directly. Seller cannot hide litigation.
Unregistered broker operates without accountability
→ DS Certified Broker program (₹5,000/year). Only certified brokers on platform. Background verified. Fraud = permanent blacklist + Karnataka REA report.
Mutation rejected, buyer stuck for years
→ DS Mutation Service reads rejection reason, prepares corrected application, refiles. Maximum 3 attempts. Government SLA tracked. Escalated to Tahsildar if delayed.
Someone transfers your property without your knowledge
→ Property Alert Monitor watches all 7 portals 24/7. Any change = SMS + push notification within 30 minutes. Owner can freeze via DigiSampatti immediately.
Bank approves loan on fraudulent property
→ Bank API integration. Loan cannot proceed unless DS verification score ≥ 70. Banks get real-time fraud status before disbursement.
Seller identity fraud (impersonation)
→ Aadhaar OTP + face match at registration. Name on Aadhaar must match RTC owner name (fuzzy match with tolerance). Discrepancy = manual review.
